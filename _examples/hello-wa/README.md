# 凹语言™（凹读音“Wa”）例子

欢迎使用 凹语言™（凹读音“Wa”） 语言: https://wa-lang.org

```
+---+    +---+
| o |    | o |
|   +----+   |
|            |
|    \/\/    |
|            |
+------------+
```

安装和测试:

1. 安装 [Clang](https://clang.llvm.org), 确保本地 `clang` 命令有效
2. `go install github.com/wa-lang/wa@latest`
4. `wa run .`
